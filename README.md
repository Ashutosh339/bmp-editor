# BMP Image Editor

A high-performance, low-level BMP image processor written in C/C++. This project focuses on binary file manipulation, efficient memory management, and systems-level programming.

## Project Goal
To build a robust tool that parses and manipulates BMP image files directly, demonstrating deep understanding of memory structures and file I/O operations—a key skill for systems engineering roles.

## Technical Details
- **Language**: C++ (with C-compatible parsing logic)
- **Focus**: Low-level binary file handling using `#pragma pack(push, 1)` to prevent memory padding.
- **Goal**: Implement header parsing, pixel data manipulation, and basic image processing filters.
